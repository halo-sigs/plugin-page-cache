# 页面静态缓存

为 Halo 2 提供页面的静态缓存功能，提高页面访问速度。

## 使用

1. 在内置的 Halo 应用市场搜索 **页面静态缓存** 或访问 <https://www.halo.run/store/apps/app-BaamQ> 手动下载并安装。
2. 启动插件之后会自动生效。

## 注意事项

1. 当前不支持监听网站内容的变动，如果网站有内容更新并需要立即展示给访客，请点击 Console 右下角的按钮手动刷新缓存。
2. 页面缓存仅对未登录的访客有效，登录用户不会使用缓存。
